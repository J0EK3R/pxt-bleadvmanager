# pxt-BleAdvertisingManager
An extension to send multiple Bluetooth Advertisements for Micro:bit using Makecode

# development with vscode 

## clone from github

'''
git clone https://github.com/J0EK3R/pxt-bleadvmanager
'''

## prepare pxt build

'''
pxt target microbit
'''

'''
pxt install
'''

'''
pxt build
'''
